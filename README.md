# Portfolio Website - HTML Version

## Overview
This is the standalone HTML/CSS/JavaScript version of Abdul Hasib's portfolio website.

## Features
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Project Showcase**: Interactive project carousel with image galleries
- **Case Study**: Detailed hotel onboarding application case study
- **Smooth Animations**: Seamless marquee text and hover effects
- **Modal Gallery**: Full-screen image viewer with keyboard navigation

## Files Structure
```
HTML version/
├── index.html                          # Main portfolio page
├── hotel-onboarding-application.html   # Case study page
├── script.js                           # JavaScript for interactivity
├── styles/
│   └── globals.css                     # Global styles and animations
└── README.md                           # This file
```

## How to Use
1. Open `index.html` in any modern web browser
2. Navigate through the portfolio projects
3. Click "Case Study" on the first project to view the detailed case study
4. Use the gallery to view all images in full-screen mode

## Technologies
- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework (via CDN)
- **Vanilla JavaScript**: For all interactive features
- **CSS3 Animations**: Smooth transitions and marquee effects

## Browser Compatibility
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes
- All images are loaded from Unsplash CDN
- No build process required - just open the HTML files
- Fully standalone - no external dependencies except Tailwind CDN
